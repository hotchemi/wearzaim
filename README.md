## Install

GGi clone and add `mobile/res/values/api_config.xml`.

```xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <string name="api_key">xxxxx</string>
    <string name="api_secret">xxxxx</string>
    <string name="callback">https://xxxx.com</string>
</resources>
```
